# Specification

The working product notes live in the repository `spec/` directory:

- `spec/spec_v2.md`: v2 product specification
- `spec/learnings.md`: container and runtime lessons from development
- `spec/codex.md`: local Codex sandbox notes

Those files are kept outside the MkDocs tree so they can remain development notes rather than polished user documentation. The published docs focus on installation, CLI behavior, host runtime setup, and pipx compatibility.
